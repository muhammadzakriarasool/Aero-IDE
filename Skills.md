# Skills

## STRICT TYPESCRIPT
All files must be strictly typed `.ts` or `.tsx`. No `any` types.

## LIGHTWEIGHT EXECUTION
Do not install heavy dependencies.

## WEBCONTAINER VITE RULE
All WebContainer scaffolding generated by the AI MUST utilize Vite + React. Never scaffold Next.js inside the WebContainer.

## UI/UX
Use Tailwind CSS v4 and Shadcn UI.
